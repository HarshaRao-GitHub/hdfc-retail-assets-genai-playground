# HDFC Retail Assets — GenAI Leadership Enablement Program

---

**Audience:** n-1, n-2 Leaders — Sales, Product, Portfolio Management, Service

**Duration:** 2 days × 8 hours | Personal laptops  
**Platform:** HDFC GenAI Playground — https://hdfc-genai-playground.vercel.app  
**Data Policy:** 100% Synthetic data only. No real customer data on any external platform.

## Session 2: Prompt Engineering + Automation — STAR SESSION (10:45–12:30)

### Content to Cover

- **The CRAFT Framework:** Context · Role · Action · Format · Target
- **Prompt Ladder Approach:** L1 (Simple) → L2 (Detailed) → L3 (Analytical) → L4 (CRAFT)
- Live demonstration: Same question, 4 different prompt levels — watch the output transform
- 3 Lab Experiments with progressive prompts
- Automation of repetitive sales tasks — MIS compilation, follow-up emails, daily reports
- Personal 30-Day GenAI adoption plan

### App Features Used

| Feature | How to Use |
|---|---|
| **Prompt Lab — Experimentation Lab** | Run all 3 experiments through L1→L2→L3→L4 levels |
| **Prompt Lab — Use-case Templates** | Browse 10 mortgage-themed prompt ladders |
| **Prompt Lab — Document Context** | Load docs via Document Tray, then prompt against them |
| **Use Case Library** | Run "Automating Repetitive Sales Activities" and "Your First 30-Day GenAI Plan" |

### User Flow — Hands-on Exercise

1. Open **Prompt Lab** → Expand **Prompt Experimentation Lab**
2. Pick **Experiment 1** (e.g., Home Loan Market Strategy)
3. Click **L1 prompt** → Send → See basic response (150-200 words, no structure)
4. Click **L2 prompt** → Send → See improved response (tables, some depth)
5. Click **L3 prompt** → Send → See analytical output (sections, comparisons)
6. Click **L4 prompt** (CRAFT) → Send → See consulting-quality output (charts, tables, executive summary, recommendations)
7. **Observe the transformation** — same topic, dramatically better output through better prompting
8. Repeat with Experiments 2 and 3
9. Switch to **Use Case Library** → Find "Automating Repetitive Sales Activities" → **Run This Use Case**
10. Finally, run "Your First 30-Day GenAI Plan" → Personalize it during the session

## Session 12: Capstone Group Activity + Presentation (4:45–6:00)

### Content to Cover

- Each team drafts and runs an AI use case plan specific to their function
- Use the **"Build Your Own Use Case"** feature end-to-end
- Teams present their use case, findings, and implementation recommendations
- Facilitator provides feedback
- Brief assessment through presentations to gauge learning

### App Features Used

| Feature | How to Use |
|---|---|
| **Build Your Own Use Case** | PRIMARY — teams define, attach docs, run, and refine |
| **Document Tray** | Upload team-specific documents |
| **Doc Intelligence** | Process any relevant documents |
| **All features** | Teams can use any feature that supports their use case |

### User Flow — Capstone Exercise

1. **Form teams** (by function: Sales, Product, Portfolio, Service)
2. Each team opens **Use Case Library** → **"Build Your Own Use Case"**
3. **Define:** Title + Business Context + Attach relevant documents
4. **Write:** A comprehensive prompt describing their use case
5. **Run:** Execute and review the AI output
6. **Refine:** Ask follow-up questions, iterate on the output
7. **Download:** Use the download buttons to save outputs
8. **Present:** 5-minute presentation per team — use case, output, and next steps
9. **Feedback:** Facilitator and peer feedback
